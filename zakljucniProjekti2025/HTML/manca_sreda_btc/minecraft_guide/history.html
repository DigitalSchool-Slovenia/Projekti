Game's history


The first public alpha of Minecraft (Then known as Cave Game) was released by Markus "Notch" Persson on 17 May 2009. The full release was on 18 November 2011, shortly after which "Notch" stepped down and Jens "Jeb_" Bergensten was named the game's lead developer.


Mojang (the developer of Minecraft) was acquired by Microsoft on the 15 September 2014 for 2.500.000.000$.



Editions


JAVA

Java edition is the original version of the game. The name was chosen because the game is written in Java. This version of the game is avaliable on PC, MacOS and Linux.


BEDROCK

This is the universal edition, avaliable on many devices, such as PC, Android, iOS, Chromebook, and Playstation, Xbox, Nintendo consoles. It is written in C++. It also enables cross-play across devices.


POCKET

This edition was made for mobile devices. It had limited worlds because of the lack of performance of early phones. It was merged with Bedrock edition in 2017.


CONSOLE

This edition was developed by a 3rd party developer named 4J Studios. The version had similar limitations as Pocket edition. Later consoles came with Bedrock edition
