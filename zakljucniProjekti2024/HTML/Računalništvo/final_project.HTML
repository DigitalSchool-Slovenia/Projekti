Cables


typs of cables

  1. Coaxial cables: cable is primarily used by cable TV companies to connect their satellite antenna facilities to customer homes and businesses.
  2. Twisted pairs: they are used in many applications in networking and communication, for example, in telephone lines, Digital Subscriber Line and local area networks.
  3. Optical fibers: they are used used to transmit telephone signals, Internet communication and cable television signals. It is also used in other industries, including medical, defense, government, industrial and commercial.
  4. Patch cables: they are used to connect ("patch in") one electronic or optical device to another for signal routing
  5. Power cables: they are used for transmission of electrical power.
  6. Data cables: they are used to transmit electronic information from a source to a destination.

Shortcuts

  1. Copy: Ctrl + C
  2. Cut: Ctrl + X
  3. Paste: Ctrl + V
  4. Maximize Window: F11 or Windows logo key + Up arrow
  5. Open Task View: Windows logo key + Tab
  6. Display and hide the desktop: Windows logo key + D
  7. Switch between open apps: Alt + Tab
  8. Open the Quick Link menu: Windows logo key + X
  9. Lock your PC: Windows logo key + L
  10. File menu: Alt + F Ctrl + F2 + E
  11. View menu: Alt + V Ctrl + F2 + V
  12. Select all: text Ctrl + A Cmd + A
  13. Copy text: Ctrl + C Cmd + C
  14. Find text: Ctrl + F Cmd + F
  15. Find and replace text: Ctrl + H Cmd + F
  16. New Document: Ctrl + N Cmd + N
  17. Open a file: Ctrl + O Cmd + O
  18. Print options: Ctrl + P Cmd + P

shortcuts are used for faster typing
